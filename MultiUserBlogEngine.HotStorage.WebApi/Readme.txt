HotStorage - stateless сервис, хранящий данные для обработки самых частых readonly-запросов (лучшее, новое, самые читаемые посты). 
Поддерживает горизонтальное масштабирование. 
Любой инстанс может быть перезапущен, остановлен без ощутимого вреда для приложения (при условии наличия других работающих инстансов HotStorage).
